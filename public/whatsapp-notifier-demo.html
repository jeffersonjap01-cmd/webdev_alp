üíª JavaScript Code

Here's how the WhatsAppNotifier class works:

// Initialize the notifier
const notifier = new WhatsAppNotifier();

// Send appointment reminder
await notifier.sendAppointmentReminder(appointmentId);

// Send appointment confirmation
await notifier.sendAppointmentConfirmation(appointmentId);

// Send prescription notification
await notifier.sendPrescriptionNotification(prescriptionId);

// Get WhatsApp Web link
const link = await notifier.getWhatsAppLink({
  phone: '081234567890',
  message: 'Hello from VetCare!'
});
